# 📝 Daily Task Tracker

A simple C++ console application to manage daily tasks, track deadlines, and monitor productivity.

## ✨ Features

- 🔐 User Registration & Login
- ➕ Add Tasks
- 👀 View Tasks
- ✅ Mark Tasks as Completed
- 🗑️ Delete Tasks
- ⏰ Deadline Tracking
- 📊 Productivity Score
- 💾 File-Based Data Storage

## 🛠️ Technologies Used

- C++
- OOP Concepts
- File Handling
- STL

## 🚀 Run the Project

```bash
g++ main.cpp -o tracker
./tracker
```

## 📂 Files

- `main.cpp` → Source Code
- `users.txt` → User Data
- `tasks_username_data.txt` → Task Data
